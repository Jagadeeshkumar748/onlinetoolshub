What is Hex to RGB Conversion?

Hex to RGB conversion is the process of translating a hexadecimal color code (used in web design and CSS) into its equivalent Red, Green and Blue component values. A hex color code like #FF5733 represents a combination of red (FF = 255), green (57 = 87) and blue (33 = 51) intensity values. Web designers, developers and graphic artists frequently need to convert between these formats for different tools and platforms.

How to Use This Converter

Enter a hex color code (with or without the # symbol) in the Hex to RGB mode, or enter individual R, G and B values (0 to 255) in the RGB to Hex mode. The converter works in real time as you type. You will see an instant color preview swatch, and the equivalent values in Hex, RGB and HSL formats. Click the Copy button next to any format to copy it to your clipboard for use in your projects.

Conversion Formula

To convert hex to RGB, each pair of hex digits is converted to its decimal equivalent:

R = parseInt(hex[1..2], 16)

G = parseInt(hex[3..4], 16)

B = parseInt(hex[5..6], 16)

Shorthand hex codes like #FFF are expanded to #FFFFFF by doubling each digit. To convert RGB to hex, each decimal value is converted to a two-digit hexadecimal string and concatenated with a # prefix.

Frequently Asked Questions

What is a hex color code?

A hex color code is a six-character string (preceded by #) that represents a color using hexadecimal values for red, green and blue channels. For example, #000000 is black and #FFFFFF is white.

What is the difference between RGB and HSL?

RGB defines colors by their red, green and blue light components. HSL defines colors by Hue (0-360 degrees), Saturation (0-100%) and Lightness (0-100%). HSL is often more intuitive for adjusting color shades and tints.

Can I enter shorthand hex codes like #FFF?

Yes. This converter automatically expands shorthand three-character hex codes. For example, #FFF becomes #FFFFFF and #F00 becomes #FF0000.
