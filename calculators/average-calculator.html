What is Mean, Median and Mode?

Mean, median and mode are three fundamental measures of central tendency in statistics. They each describe the centre of a data set in different ways:

Mean (Average): The sum of all values divided by the number of values. For example, the mean of 2, 4, 6 is (2+4+6)/3 = 4. The mean is sensitive to extreme values (outliers).

Median: The middle value when all numbers are arranged in ascending order. If there is an even number of values, the median is the average of the two middle values. For example, the median of 1, 3, 5, 7 is (3+5)/2 = 4. The median is less affected by outliers than the mean.

Mode: The value that appears most frequently in the data set. A data set can have one mode (unimodal), multiple modes (multimodal), or no mode if all values appear with equal frequency.

When to Use Which Measure

Use Mean when: The data is evenly distributed without extreme outliers. It is the most commonly used average and works well for symmetric distributions like test scores or heights.

Use Median when: The data has outliers or is skewed. For example, median income is more representative than mean income because a few extremely wealthy individuals can skew the mean upward.

Use Mode when: You want to find the most common or popular value. It is useful for categorical data, such as the most popular shoe size or the most frequently ordered dish.

Other Statistics Provided

In addition to mean, median and mode, this calculator also provides:

Sum: The total of all values added together.

Count: The number of values in the data set.

Min: The smallest value in the data set.

Max: The largest value in the data set.

Range: The difference between the maximum and minimum values (Max - Min). A larger range indicates more spread in the data.

Applications in Statistics and Everyday Life

These measures are used across many fields. Teachers use the mean to calculate class averages. Businesses use the median to understand typical customer spending. Fashion retailers use the mode to stock the most popular sizes. Scientists use all three measures to analyse experimental data. Financial analysts use them to understand market trends and investment returns.

Frequently Asked Questions

What if there is no mode?

If all numbers in the data set appear the same number of times, there is no mode. This calculator will display "No mode" in such cases. For example, the set 1, 2, 3, 4, 5 has no mode because each value appears exactly once.

Can there be more than one mode?

Yes. If two or more values share the highest frequency, the data set is multimodal. For example, in the set 1, 2, 2, 3, 3, 4, both 2 and 3 are modes. This calculator will display all modes.

How do I enter decimal numbers?

You can enter decimal numbers using a dot as the decimal separator. For example: 1.5, 2.7, 3.14, 4.0. Separate values with commas or spaces.
